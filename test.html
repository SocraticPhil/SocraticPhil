Some common block tags

hr

The "hr" element used to stand for "horizontal line" and that is still how it is typically displayed by browsers. However, in HTML5 the tag took on a semantic meaning, "semantic break." I will be using it often to break up example files when when I want to display multiple tags in one file.

address

The "address" element indicates that the text is an address specific to the page, perhaps contact information for the author.
1715 Walnut Blvd
Ashtbula OH 44004

blockquote

The "blockquote" element is used for extended quotations. It should include the cite attribute to show or link to the source of the quote.
"If you can't do great things, do small things in a great way." Napolean Hill.
